# Wasmito

### Dependencies

We depend on the folliwing libraries:
- python3-serial
- cargo
- cargo bins
- node

### Installation

After installing the dependencies run the installation script. This will install all the CLI tool.
> bash scripts/install.sh

After installation, you can use the CLI as follow:

> node dist/cjs/cli/cli.cjs


create an alias


### Example

```
> wasmito callgraph example.wasm
```
