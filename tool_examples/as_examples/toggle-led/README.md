# Toggle LED

The `toggl-led` directory contains an AssembylScript application that toggles an LED when pressing buttons.
The application was written to target the M5StickC pin layout and can be changed to target other ESP32 boards.

The `toggle-led` project directory contains the following files:
- `src/arduino.ts`: an AS source file that abstracts the use of the WARDuino primivites.
- `src/toggle-led.ts`: the AS source file that uses the `arduino.ts` and implements the toggling behaviour.
- `wasm/toggle-led.wasm`: the Wasm module obtained after compiling the `toggle-led.ts`
- `wasm/mappings.json`: the extract mappings from the Source Map Spec debugging information generated by the AssemblyScript compiler. The creation of this file is optional but recommended as it eases the creation of tools for AssemblyScript. See tutorials for an explanation on how to generate this file.


The `toggle-led.wasm` was compiled using AssemblyScript `Version 0.27.5` using the following command:
```bash
asc src/toggle-led.ts --sourceMap --debug -o wasm/toggle-led.wasm
```
